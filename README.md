# NeoWiki

[NeoWiki](https://professional.wiki/en/neowiki) is a collaborative knowledge management system on top of MediaWiki and graph databases.

## Technical Documentation

See [docs/](./docs/), especially [docs/Glossary.md](./docs/Glossary.md).

## Development

### NeoWiki frontend

Initial setup: run `make ts-install ts-build` to install the dependencies and build the frontend code.

For other actions, refer to the `ts-*` commands in the [Makefile](./Makefile).
